XHTMLL & CSS

XHTML and CSS are standard code that make websites work
By right clicking in the windo and selecting view source you can view the code behind any website
It looks complicated at first, but it is actually easy
Websites are just text files

Tags

XHTML revolves around tags
Every component is contained around a tag
The first one declares the type of file the document will be

Scope

Most tags have openings and closings
Everything outside a tag belongs to another object

Head

 The head of the site is the information for the browser
The information in the head isn't seen by the user
Includes key words for the search engines, title of the site, browser information and more

Body

 The body of the site includes the sites content
It includes paragraphs, images, audio, animations, backgrounds and more
There are tags for paragraphs, line breaks, horizontal rule(horizontal line) bold and italics

Comments

 You can comment in HTML just like in python and other languages
You can also add links to pages with an anchor tag

Header

 You can add headers to the sight
The tags for this are h1, h2, h3, h4, h5, h6
h1 is the biggest and h6 is the smallest
